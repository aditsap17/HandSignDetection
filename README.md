# Hand Sign Detection A–Z

Project UAS pendeteksian bahasa isyarat A–Z
menggunakan OpenCV, MediaPipe, dan Flask.

## Cara Menjalankan

1. Install library
2. Jalankan dataCollection.py
3. Hasil sample akan masuk ke folder A-Z
4. Training sample pada website teacheblemachine
5. Masukkan file modelnya yang sudah ditraining ke folder Model
6. Jalankan app.py
7. Buka browser: http://127.0.0.1:5000
